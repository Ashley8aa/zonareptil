Dispositivo

Este sitio web está recibiendo datos en base una placa ESP32, utilizando sensores y actuadores. Sus funciones son que censa la temperatura y humedad de un terrario de reptiles. Esto es posible, ya que la placa ESP32 tiene conexión Wifi y con ello se muestran los datos obtenidos en nuestro sitio web.

Sitio

Este sitio web está desarrollado con herramientas tales como angular para la interfaz y Mongodb para administrar la base de datos. También se utiliza arduino para conectar el dispositivo con el sitio web y la base de datos.

Nosotros

Somos una empresa pequeña que apenas está comenzando, con tan solo 3 desarrolladores. Ofrecemos el servicio de nuestro nuevo desarrollo “Zona reptil”, el cual es un sitio multiplataforma con una arquitectura muy fácil de utilizar.
